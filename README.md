
# Sensor NTC-10K |arduino + Streamlit🤖


projeto integrador Univesp - Engenharia da computação

-Arduino Uno, sensor de temperatura, código python coletando dados e gerando graficos em tempo real 

## Streamlit🤖 |plotly

-visualizador.py :carrega um grafico com parametros de 0 a 70ºC
-visualizador 2.py : carrega o grafico sem parametros porém escalável de acordo com a temperatura 

um arquivo CSV é gerado para que seja possivel realizar uma analise dos dados.
